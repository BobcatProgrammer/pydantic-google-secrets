============
Installation
============


Stable release
--------------

To install Pydantic Google Secrets, run this command in your terminal:

.. code-block:: console

    $ pip install pydantic_google_secrets

This is the preferred method to install Pydantic Google Secrets, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From source
-----------

The source for Pydantic Google Secrets can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/BobcatProgrammer/pydantic_google_secrets

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/BobcatProgrammer/pydantic_google_secrets/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ pip install .

.. _Github repo: https://github.com/BobcatProgrammer/pydantic_google_secrets
.. _tarball: https://github.com/BobcatProgrammer/pydantic_google_secrets/tarball/master
